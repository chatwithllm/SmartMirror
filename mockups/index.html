Smart Mirror · design mockups

Two collections. Style directions (01–05) show the same data in five visual languages — pick the one that fits. Mode mockups (06–10) show the same chosen language applied to five full dashboards — morning, work, relax, shopping, security — each with live clocks, clickable lists, scrub bars, and real demo state.

Target: 43" TV mounted vertical · 9:16 portrait · 4K (2160 × 3840). Mockups are authored at 1080 × 1920 base so they render on any laptop; production build scales to 4K with the same grid and ships a runtime resolution switcher (see DESIGN_SPEC §4). Self-contained HTML, no build step — open each file directly for full fidelity and interactivity.
